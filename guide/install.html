安装 #

环境支持 #

支持js和ts,所以可以在vue2,vue3等环境的项目中使用它。

版本 #

mmqUtils 目前发布了第一个大版本1.0.0,接下来会持续更新。

使用包管理器 #

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 mmqUtils, 然后您就可以使用打包工具，例如 Vite 和 webpack等

# 选择一个你喜欢的包管理器

# NPM
$ npm install mmq-utils --save

# Yarn
$ yarn add mmq-utils

# pnpm
$ pnpm install mmq-utils
